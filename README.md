# DevTools Plus

> A modern, privacy-first browser extension that bundles 20+ everyday developer tools into one place.

🚧 **Project bootstrapping in progress.** The full scaffold and first set of tools are being generated automatically.

## Vision

- 🔒 **100% offline** — no servers, no tracking, no LLM API calls
- ⚡ **One-click access** — popup, side panel, and right-click menu
- 🎨 **Modern UI** — built with Tailwind + shadcn/ui, dark mode included
- 🌍 **i18n** — English & 中文 from day one
- 💝 **Free forever** — supported by donations

## Planned tools

JSON Formatter · Base64 · UUID · Regex Tester · JWT Decoder · Hash · URL Encode · Timestamp · Cron Parser · Color Converter · and more.

See `docs/ROADMAP.md` (coming with the first PR).
